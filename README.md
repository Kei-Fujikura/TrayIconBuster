# TrayIconBuster
A utility that removes phantom icons from the Icon Tray.

# What is this.
This is copy project of LP-TrayIconBuster.
https://www.codeproject.com/Articles/19620/LP-TrayIconBuster

# Change point
Currently, the following changes have been made.
* Suppressed multiple launch of exe
* Continues to work if it fails due to some error during execution
